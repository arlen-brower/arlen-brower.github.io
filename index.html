Requirements to Outcomes

Let's take section 3.3 as an example

  • Clear the terminal screen
    • ... use system('clear')
  • Print the 2D map
    • ... use nested for loops on an array
  • Update the map with each input
    • ...do this in a modular way

Example Submission


						void renderMap(int rowCount, int columnCount, int playerRow, int playerColumn, int goalRow, int goalColumn,
						char** mapArray)
						{
						int row, column;

						/* Clear the terminal screen */
						system("clear");

						/* Print out the top border of the map */
						renderHorizontalMapEdge(columnCount);

						/* Loop through every row that needs to be printed out, making sure to include the map edge */
						for (row = 0; row < rowCount; row ++) { /* Left edge of the row */ putchar(MAPEDGE); putchar(MAPSPACE); /*
							Now loop through the row and print out each character in the character array, but also checking for the
							goal and the player */ for (column=0; column < columnCount; column ++) { if (row==playerRow &&
							column==playerColumn) { putchar(PLAYER); } else if (row==goalRow && column==goalColumn) { putchar(GOAL); }
							else { putchar(mapArray[row][column]); } putchar(MAPSPACE); } putchar(MAPEDGE); putchar('\n'); }
